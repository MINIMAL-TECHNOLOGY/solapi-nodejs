    • Defined in src/solapi.ts:85

    메시지 발송 기능, sendMany 함수에서 조금 더 개선된 오류 표시 기능등을 제공합니다. 한번의 요청으로 최대 10,000건까지 발송할 수 있습니다.

    MessageNotReceivedError 메시지가 모두 발송 접수가 불가한 상태일 경우 MessageNotReceivedError 예외가 발생합니다.

    Parameters

    • messages: Message | Message[]

      발송 요청할 메시지 파라미터(문자, 알림톡 등)

    • Optional scheduledDate: string | Date

      예약일시

    • allowDuplicates: boolean = false

      중복 수신번호 허용 여부

    • Optional appId: string

      appstore용 app id

    Returns Promise<DetailGroupMessageResponse>
